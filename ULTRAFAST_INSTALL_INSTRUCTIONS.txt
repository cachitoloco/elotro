═══════════════════════════════════════════════════════════════════════════════

📦 STEP 2: EXTRACT THE PACKAGE

tar -xzf worldchain-bot-ultrafast-v2.1.tar.gz
cd worldchain-bot-ultrafast-v2.1/

═══════════════════════════════════════════════════════════════════════════════

⚙️ STEP 3: INSTALL DEPENDENCIES

# Install Node.js dependencies
npm install

# Install HoldStation SDK (required for ultra-fast execution)
chmod +x install-holdstation-sdk.sh
./install-holdstation-sdk.sh

═══════════════════════════════════════════════════════════════════════════════

🔧 STEP 4: CONFIGURE SETTINGS

# Copy configuration template
cp .env.example .env

# Edit configuration (add your private keys and API keys)
nano .env

Required settings in .env:
- PRIVATE_KEY=your_wallet_private_key
- WORLDCHAIN_RPC_URL=your_rpc_endpoint
- ALCHEMY_API_KEY=your_alchemy_key

═══════════════════════════════════════════════════════════════════════════════

🚀 STEP 5: START ULTRA-FAST TRADING

# Start the bot
node worldchain-trading-bot.js

# Or run in background
nohup node worldchain-trading-bot.js > bot.log 2>&1 &

🛠️ TROUBLESHOOTING:

Problem: "Cannot find module" errors
Solution: Run "npm install" in the bot directory

Problem: HoldStation SDK errors
Solution: Run "./install-holdstation-sdk.sh"

Problem: Slow execution times
Solution: Use "Enhanced Trade" (option 1) and check RPC endpoint

Problem: Configuration errors
Solution: Check .env file settings and ensure all keys are added
