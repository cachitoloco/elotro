📦 QUICK INSTALLATION:

1. DOWNLOAD & EXTRACT:
   wget https://github.com/romerodevv/psgho/raw/main/worldchain-bot-ultrafast-v2.2.tar.gz
   tar -xzf worldchain-bot-ultrafast-v2.2.tar.gz
   cd worldchain-bot-ultrafast-v2.2/

2. INSTALL DEPENDENCIES:
   npm install

3. CONFIGURE:
   cp .env.example .env
   nano .env  # Add your private key and RPC endpoint

4. START BOT:
   node worldchain-trading-bot.js

🎯 NEW STRATEGY BUILDER WORKFLOW:

1. Select option 5 from main menu: "Strategy Builder"
2. Create new strategy with your preferred:
   - Token pair (from discovered tokens)
   - DIP threshold (e.g., 15% price drop)
   - Profit target (e.g., 3% profit)
   - Trade amount (e.g., 0.1 WLD)
3. Start strategy - it will WAIT for DIP before buying
4. Monitor real-time progress and position management
5. Automatic sell when profit target reached

⚠️  IMPORTANT: DIP strategies do NOT buy immediately!
   They monitor prices and wait for the specified drop.
